Calculator
A simple, interactive calculator built using HTML, CSS, and JavaScript.

🚀 Features
Clean UI: Built with CSS for a modern, responsive look.

Core Functionality: Performs all standard arithmetic operations (addition, subtraction, multiplication, and division).

Interactive: Powered by JavaScript to handle user inputs and calculations instantly.

🛠️ Technologies Used
HTML5: Structure of the calculator.

CSS3: Styling and layout.

JavaScript: Logic for calculations and button interactivity.

💻 How to run this project
Clone this repository to your local machine:

Bash
git clone https://github.com/kumarmanjeet7479-jpg/Calculator.git
Open the index.html file in your preferred web browser.
